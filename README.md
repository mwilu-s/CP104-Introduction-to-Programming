# CP104 – Introduction to Programming

**Wilfrid Laurier University**  
**Language:** Python 3 | **Tools:** Eclipse / PyDev

---

## About This Course

CP104 is an introductory programming course focused on problem-solving and structured program design. Topics covered include:

- Variables, data types, and expressions
- Input/output and string formatting
- Conditional statements and boolean logic
- Loops (`for` and `while`)
- Functions, parameters, and return values
- Lists, tuples, and file I/O
- Recursion and modular design

All code is written in Python 3 and follows the [CP104 coding and documentation standards](https://bohr.wlu.ca/cp104/standards.php), including PEP8 style, descriptive naming conventions, and structured docstrings.

### Labs (`labs/`)
11 labs completed throughout the course. Labs are autograded and focus on applying weekly concepts through guided tasks.

### Assignments (`assignments/`)
9 assignments that build on lab concepts with less scaffolding. Assignments are submitted for manual grading and demonstrate independent problem-solving.

## Code Style

All Python files follow the CP104 documentation standard:

```python
"""
-------------------------------------------------------
[Description of the program or module]
-------------------------------------------------------
Author: [Name]
ID:     [Student ID]
Email:  [Email]
__updated__ = "YYYY-MM-DD"
-------------------------------------------------------
"""
```

Functions include structured docstrings with parameter types, constraints, and return value descriptions.
